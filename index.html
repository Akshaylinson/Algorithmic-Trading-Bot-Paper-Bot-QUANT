Project Architecture

Modular design with clear separation of concerns for maintainability and extensibility.

Project Structure

Algorithmic-Trading-Bot/
├── quantbot/
├── broker.py # Handles paper/live trading via Alpaca
├── utils.py # Fetches OHLC data (Alpaca/yFinance)
├── strategy.py # Technical indicators and logic
└── runner.py # Command-line bot runner
├── streamlit_app.py # Interactive dashboard
├── run_bot.py # CLI script for trading loop
└── requirements.txt # Python dependencies

Get Started in Minutes

Follow these simple steps to set up the Algorithmic Trading Bot on your machine.

Quick Setup Guide

1

Clone the repository

git clone https://github.com/your-username/Algorithmic-Trading-Bot.git
2

Create and activate virtual environment

python -m venv venv
venv\Scripts\activate # Windows
source venv/bin/activate # Mac/Linux
3

Install dependencies

pip install -r requirements.txt
4

Set up environment variables

# Create .env file with:
MODE=ALPACA
ALPACA_KEY=your_alpaca_api_key
ALPACA_SECRET=your_alpaca_secret_key
ALPACA_BASE_URL=https://paper-api.alpaca.markets

Run CLI Trading Bot

python run_bot.py

Starts the continuous trading loop and executes simulated/paper trades automatically.

Launch Streamlit Dashboard

streamlit run streamlit_app.py

Opens the interactive dashboard at http://localhost:8501
